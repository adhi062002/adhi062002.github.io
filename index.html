01 — Multi-Sensor Fusion2024 – present
Autonomous UGV Localization via Sensor Fusion
CSIR-National Aerospace Laboratories
+ expand details
Integrated visual-inertial odometry (VINS-Fusion) with 2D LiDAR and RGB-D sensing for GPS-denied navigation. Designed and implemented an error-state Extended Kalman Filter for pose fusion, with LiDAR-based pose correction and depth projection. Sensor synchronization and coordinate frame alignment handled via ROS2 TF. Evaluated using ATE and RPE metrics; architecture structured for future factor-graph migration.

02 — Computer Vision2024 – present
YOLO-Based Real-Time Dent Detection System
CSIR-National Aerospace Laboratories
+ expand details
Built real-time defect detection pipeline for aircraft surface inspection using YOLO (v5/v8). Developed a dual-model YOLOv8 framework with cross-model Non-Maximum Suppression, eliminating rivet-vs-dent misclassification and achieving ~88% accuracy. Modified YOLOv8 pipeline with CLAHE + Laplacian + HSV normalization reached 96.11% at conf 0.3. Validated on composite, metal, and FML specimens across varied lighting conditions.
